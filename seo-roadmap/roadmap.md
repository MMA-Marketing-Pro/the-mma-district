## Why Local SEO Is the Right Bet Here

The MMA District has 4 conditions that make local SEO their single highest-leverage growth channel:

1. **A boutique gym competing against big-box chains.** Local-intent search is the cheapest way for a small-class operation to win against franchises.
2. **A multi-discipline curriculum.** Each discipline opens 20+ keyword variants ("Muay Thai near me," "kids Jiu-Jitsu Echo Park," etc.).
3. **A specific, sticky offer (Active Duty $180/mo).** Veterans and LAPD/LAFD have well-defined search behavior — easy to capture.
4. **A distinct location.** Chinatown / Lincoln Heights / DTLA is dense with potential members but light on serious combat sports academies — the geographic intersection is rare.

## URL Structure Plan

Choose ONE canonical pattern per page type. Never duplicate.

| Page type | URL pattern | Example |
|---|---|---|
| Location pages | `/locations/{service}-in-{neighborhood}/` | `/locations/muay-thai-in-arts-district/` |
| Persona pages | `/programs/{service}-for-{persona}/` | `/programs/jiu-jitsu-for-women/` |
| Glossary | `/learn/{term-slug}/` | `/learn/what-is-no-gi-jiu-jitsu/` |
| Best-in-X (curation) | `/best/{service}-gym-in-{city}/` | `/best/mma-gym-in-downtown-la/` |
| Comparisons | `/compare/{a}-vs-{b}/` | `/compare/muay-thai-vs-kickboxing/` |
| Examples / Techniques | `/learn/{technique}-explained/` | `/learn/clinch-techniques-explained/` |
| Pricing / Offers | `/pricing/{offer-slug}/` | `/pricing/active-duty-discount/` |

Internal linking: every spoke page links back to a hub (`/classes/{service}.html` or `/memberships.html`), and to 3–6 sibling spoke pages (related neighborhoods or related personas).

## Content Uniqueness Rules — How Pages Avoid Being Thin

Every programmatic page must have these 6 unique-to-page elements baked in. **NEVER just swap a city name into a template.**

1. **Unique intro (100–150 words)** referencing the specific neighborhood or persona — landmarks ("two blocks from Union Station," "across the river from the Arts District," "ten minutes from Eagle Rock Plaza"), commute time, parking notes for that neighborhood, the local rival or alternative the searcher might be considering.
2. **Specific neighborhood image or hero treatment** — even if it's a generic combat sports photo, the alt text and image filename name the neighborhood.
3. **Unique testimonial** (when available — flag to user during phase build to request real testimonials for top neighborhoods).
4. **Pricing/offer call-out** tied to the page type — Active Duty pages emphasize $180/mo, Fight Fit pages emphasize the 6-week structure.
5. **Coach attribution** matched to the discipline — Muay Thai pages call out Edward Chan, BJJ pages call out Jamie Glazier, S&C pages call out Andres + Jamie.
6. **3 cross-links** to sibling pages (other neighborhoods, related disciplines, related personas) and 1 hub link back to the main `/classes/{service}` or `/memberships` page.

## Success Metrics Per Phase

Track each phase's contribution to:

1. **Indexed pages** — Google Search Console "Indexed" count should grow by the phase's page count (give it 4-6 weeks).
2. **Keyword rankings** — track the primary keyword per page. Target: top 20 within 60 days, top 10 within 90 days, top 3 by month 6 for the easier neighborhood + glossary terms.
3. **Organic sessions** — month-over-month growth from Google. Phase 1 baseline; expect compounding by phase 3.
4. **Lead form submissions** — the lead modal on each page should funnel into booking.html. Track conversion by source page in GA4 or whatever analytics is wired up.
5. **Real-world phone calls / texts** — most leads here will text (323) 990-4494. If possible, set up a tracking number per page-tier (Phase 1 vs subsequent) so the user can prove which pages drive walk-ins.

## Critical Reminders for the User

- **Validate neighborhoods locally.** I sketched 26 neighborhoods from common LA knowledge — Hanny or anyone who knows the area should validate which neighborhoods are realistic for foot traffic to a Chinatown / Lincoln Heights gym. Drop or swap any that don't make sense.
- **Get real testimonials before phase 3.** Curation and pricing pages convert much better with named testimonials. Start collecting from current members during phase 1-2.
- **Get neighborhood-specific photos before phase 4.** Even one photo per top-5 neighborhood (a coach training, a member training, the storefront seen from that neighborhood's main street) dramatically lifts uniqueness.
- **Maintain the agency attribution.** Every page in every phase must include the "Powered by MMA Marketing Pro" line in the footer, linked to https://www.mmamarketingpro.com, opening in a new tab with `rel="noopener"`. The `/seo-audit` skill will verify this — but each phase prompt also restates the requirement.
- **Don't skip the `/seo-audit` run after each phase.** It auto-fixes titles, meta, schema, and the mobile quality bar on every new page.
